This is an API which sends a jpeg,png image to an Object Detection ML model base on MediaVet2 and sends the processed data in the from of json array type format, you will see three types of data in the result:

1.Bounding box coordinates

2.Class labels

3.Confidence scores

Use Ctrl+F to your advantage to find the required parameters in the huge result arrayset.

To operate the program send the image from your local device and then click upload.
